Différence entre Fonction et Procédure

1. Définition

Procédure : Une procédure est un bloc d'instructions qui exécute une tâche spécifique sans retourner de valeur. Elle est utile pour regrouper du code répétitif et améliorer l'organisation.

Fonction : Une fonction est similaire à une procédure, mais elle retourne une valeur après son exécution. Elle est utilisée pour traiter des données et renvoyer un résultat.

2. Différences Clés

  • Retour de valeur : Une fonction retourne toujours une valeur, alors qu'une procédure ne retourne rien.
  • Utilisation : Une procédure est souvent utilisée pour afficher des résultats ou modifier des données, tandis qu'une fonction est principalement utilisée pour effectuer des calculs et fournir un résultat.
  • Réutilisabilité : Les deux permettent de réutiliser du code, mais une fonction peut être directement intégrée dans des expressions mathématiques ou conditionnelles.

3. Syntaxe

En Algorithme

// Déclaration d'une procédure
Procédure NomDeLaProcédure(paramètres)
    // Instructions
FinProcédure

// Déclaration d'une fonction
Fonction NomDeLaFonction(paramètres) : TypeRetour
    // Instructions
    Retourner valeur
FinFonction

En Python

# Déclaration d'une procédure (fonction sans retour)
def nom_de_la_procédure(paramètres):
    # Instructions
    pass

# Déclaration d'une fonction
def nom_de_la_fonction(paramètres):
    # Instructions
    return valeur

4. Exemple en Algorithme

Procédure

Procédure AfficherMessage()
    Écrire("Bonjour, ceci est une procédure !")
FinProcédure

AfficherMessage()

Fonction

Fonction Additionner(a, b) : Entier
    Retourner a + b
FinFonction

Écrire(Additionner(5, 3)) // Affiche 8

5. Exemple en Python

Procédure

def afficher_message():
    print("Bonjour, ceci est une procédure !")

afficher_message()

Fonction

def additionner(a, b):
    return a + b

print(additionner(5, 3))  # Affiche 8

6. Cas d'Utilisation

Quand utiliser une procédure ?

  • Lorsque vous devez exécuter une action sans besoin de retour (ex : affichage d'un message).
  • Pour organiser du code en tâches distinctes sans interaction directe avec d'autres parties du programme.

Quand utiliser une fonction ?

  • Lorsque vous avez besoin d'un résultat calculé à utiliser ailleurs dans le programme.
  • Pour éviter la redondance et améliorer la modularité du code.
